### Конструкторы
Название конструктора Всегда совпадает с названием класса


### Цели:
1. создать обьект содержащий информацию о человеке (имя, фамилия, возраст, вес, рост)*
2. просить пользователя поочередно вводить данные
2. Использовать сеттеры и геттеры 
3. Использовать полиморфизм
4. Использовать наследование
5. Использовать многоуровневую инициализацию
5. использовать блок try-catch
6. собрать всех людей в один список и вывести на экран
