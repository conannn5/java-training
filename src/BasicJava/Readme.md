### JVM
**(Java Virtual Machine)** - Это программа-интерпретатор байт кода,
позволяющая запускать программу на целевом компьютере

### JRE
**(Java Runtime Envinroment)** - это совокупность JVM,
стандартной библиотеки и прочих инструментов,
необходимых для исполнения программы

### JDK
**(Java Development Kit)** - это набор инструментов разработки,
которые нужны для программирования на Java

### Скобки и знаки
1. **()** - Аргументы
2. **{}** - Тело метода
3. **[]** - Массив
4. **&&** - И (касается условий в условном операторе*)
5. **||** - ИЛИ (или дно или второе условие истина*)
6. **=** - Присваивание значений
7. **==** - Проверка на равенство (Примитивный тип данных*)
8. **!=** - Проверка на неравенство (Тоже примитивный*)
9.

### модифакаторы доступа:
1. public            -(Общий доступ, доступно всем)
2. private           -(частный доступ, виден только внутри класса)
3. protected         -(Защищенный доступ, виден внутри одного пакета и всем наследникам)
4. Без модификатора  -(Виден всем классам внутри пакета, но не доступен снаружи)

### Класс formatter
Спецификаторы формата:
%a Шестнадцатеричное значение с плавающей точкой
%b Логическое (булево) значение аргумента
%c Символьное представление аргумента
%d Десятичное целое значение аргумента
%h Хэш-код аргумента
%e Экспоненциальное представление аргумента
%f Десятичное значение с плавающей точкой
%g Выбирает более короткое представление из двух: %е или %f
%o Восьмеричное целое значение аргумента
%n Вставка символа новой строки
%s Строковое представление аргумента
%t Время и дата
%x Шестнадцатеричное целое значение аргумента
%% Вставка знака %

### Флаги формата

Флаги формата
- Выравнивание влево
# Изменяет формат преобразования
0 Выводит значение, дополненное нулями вместо пробелов. Применим только к числам.
Пробел Положительные числа предваряются   пробелом
+ Положительные числа предваряются знаком +. Применим только к числам.
, Числовые значения включают разделители групп. Применим только к числам.
( Отрицательные числовые значения заключаются в скобки. Применим только к числам.
