### JVM
**(Java Virtual Machine)** - Это программа-интерпретатор байт кода,
позволяющая запускать программу на целевом компьютере

### JRE
**(Java Runtime Envinroment)** - это совокупность JVM,
стандартной библиотеки и прочих инструментов,
необходимых для исполнения программы

### JDK
**(Java Development Kit)** - это набор инструментов разработки,
которые нужны для программирования на Java

### Скобки и знаки
1. **()** - Аргументы
2. **{}** - Тело метода
3. **[]** - Массив
4. **&&** - И (касается условий в условном операторе*)
5. **||** - ИЛИ (или дно или второе условие истина*)
6. **=** - Присваивание значений
7. **==** - Проверка на равенство (Примитивный тип данных*)
8. **!=** - Проверка на неравенство (Тоже примитивный*)

### модифакаторы доступа:
1. public            -(Общий доступ, доступно всем)
2. private           -(частный доступ, виден только внутри класса)
3. protected         -(Защищенный доступ, виден внутри одного пакета и всем наследникам)
4. Без модификатора  -(Виден всем классам внутри пакета, но не доступен снаружи)

### Класс formatter
Спецификаторы формата:
1. %a Шестнадцатеричное значение с плавающей точкой
2. %b Логическое (булево) значение аргумента
3. %c Символьное представление аргумента
4. %d Десятичное целое значение аргумента
5. %h Хэш-код аргумента
6. %e Экспоненциальное представление аргумента
7. %f Десятичное значение с плавающей точкой
8. %g Выбирает более короткое представление из двух: %е или %f
9. %o Восьмеричное целое значение аргумента
10. %n Вставка символа новой строки
11. %s Строковое представление аргумента
12. %t Время и дата
13. %x Шестнадцатеричное целое значение аргумента
14. %% Вставка знака %

### Флаги формата

Флаги формата
1. - Выравнивание влево
2. # Изменяет формат преобразования
3. 0 Выводит значение, дополненное нулями вместо пробелов. Применим только к числам.
4. Пробел Положительные числа предваряются   пробелом
5. + Положительные числа предваряются знаком +. Применим только к числам.
6. , Числовые значения включают разделители групп. Применим только к числам.
7. ( Отрицательные числовые значения заключаются в скобки. Применим только к числам.
