### JVM
**(Java Virtual Machine)** - Это программа-интерпретатор байт кода,
позволяющая запускать программу на целевом компьютере

### JRE
**(Java Runtime Envinroment)** - это совокупность JVM,
стандартной библиотеки и прочих инструментов,
необходимых для исполнения программы

### JDK
**(Java Development Kit)** - это набор инструментов разработки,
которые нужны для программирования на Java

### Скобки и знаки
1. **()** - Аргументы
2. **{}** - Тело метода
3. **[]** - Массив
4. **&&** - И (касается условий в условном операторе*)
5. **||** - ИЛИ (или дно или второе условие истина*)
6. **=** - Присваивание значений
7. **==** - Проверка на равенство (Примитивный тип данных*)
8. **!=** - Проверка на неравенство (Тоже примитивный*)

### модифакаторы доступа:
1. public            -(Общий доступ, доступно всем)
2. private           -(частный доступ, виден только внутри класса)
3. protected         -(Защищенный доступ, виден внутри одного пакета и всем наследникам)
4. Без модификатора  -(Виден всем классам внутри пакета, но не доступен снаружи)
